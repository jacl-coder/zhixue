# 智学奇境 - 基于AI的智能学习游戏平台

**EduVerse AI - Intelligent Learning Game Platform**

---

### 🎯 项目概述
传统教育缺乏个性化和参与度，本项目结合AI技术与游戏化设计，打造智能学习平台。通过2.5D游戏界面和AI算法，实现个性化学习路径规划、难度自适应调节，提升学习效率和体验。

### 💻 技术栈

#### 🎮 客户端技术
**Unity 2022.3 LTS + C#**
- 2.5D游戏界面，兼顾视觉效果与开发效率
- Unity UI Toolkit现代化界面系统
- 跨平台部署，支持PC和移动端
- C# .NET框架，开发效率高
- Serilog 日志系统，支持异步日志记录

#### 🚀 后端服务
**Golang + Nano + Gin框架**
- **Gin框架** - RESTful API服务，高性能HTTP处理
- **Nano框架** - 游戏服务器，实时通信和会话管理
- **GORM** - 数据库ORM，简化数据操作
- **JWT认证** - 安全的用户身份验证
- **Redis集成** - 缓存和会话存储
- **Zap日志** - 高性能结构化日志系统，支持日志轮转

#### 🧠 AI推理服务
**Python + PyTorch生态**
- **FastAPI** - 异步API框架，高性能AI服务
- **PyTorch** - 深度学习模型训练和推理
- **Scikit-learn** - 传统机器学习算法
- **Pandas/NumPy** - 数据处理和分析
- **TorchServe** - 生产级模型部署
- **Loguru** - 现代化日志系统，零配置高性能

#### 🗄️ 数据存储
**PostgreSQL + Redis双重存储**
- **PostgreSQL** - 主数据库，用户数据、学习记录
- **Redis** - 缓存层，会话数据、实时状态
- **数据备份** - 自动备份和恢复机制

## 🎮 核心功能模块

### 1. 🧠 AI智能系统
- **自适应难度调节** - 基于学习表现实时调整题目难度
- **个性化路径规划** - AI推荐最优学习顺序和内容
- **学习行为分析** - 识别学习模式，预测学习效果
- **知识图谱构建** - 动态构建个人知识结构图

### 2. 🎯 游戏化学习体验
- **任务驱动学习** - 知识点包装成趣味游戏任务

- **成就激励系统** - 徽章、排行榜等游戏化激励
- **社交学习** - 好友系统、学习小组功能

### 3. 📊 智能数据分析
- **实时数据收集** - 学习行为、时间模式、正确率追踪

- **学习报告生成** - 个性化学习分析和建议报告
- **预测模型** - 学习效果预测和风险预警

### 4. 🌐 多端协同管理
- **教师管理端** - Vue.js Web平台，课程和学生管理
- **学习分析仪表板** - 实时数据可视化和报告
- **家长监控** - 学习进度和效果反馈系统
